Boltz web app

2023-02-19

introduction

around this time i started working on the Boltz web app. the goal was to make a clean browser app for non-custodial swaps between bitcoin layers.

my first public commits there were on february 19, 2023. it started with layout, navigation, build setup, assets and early swap screens, and later became the official web app served at boltz.exchange.

submarine swap UX

one small UX win i still like was saving one step in the submarine swap flow. instead of asking the user to enter an amount and then paste a lightning invoice, the app can use the amount from the pasted invoice.

that sounds small, but it matters. less typing, fewer mismatches, and one less place where the user can make a mistake while moving between bitcoin and lightning.
